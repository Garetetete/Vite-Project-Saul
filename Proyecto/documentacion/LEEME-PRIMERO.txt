📋 INICIO RÁPIDO (Elige una opción):

  1️⃣  Windows PowerShell:
      .\iniciar-proyecto.ps1

  2️⃣  Manual:
      npm install
      npm start
      
  3️⃣  Abrir en navegador:
      http://localhost:4200

───────────────────────────────────────────────────────────

🚀 PARA DESPLEGAR EN NETLIFY:

  Opción A - Rápida (Drag & Drop):
      1. Ejecutar: .\construir-para-netlify.ps1
      2. Ir a: https://app.netlify.com/drop
      3. Arrastrar carpeta: dist/mi-app/browser
      
  Opción B - Automática (Git):
      1. Subir a GitHub
      2. Conectar con Netlify
      3. Auto-deploy configurado con netlify.toml

✨ CARACTERÍSTICAS DEL PROYECTO:

  ✅ Consume 3 endpoints de Cat Facts API:
     • GET /fact .............. Hecho aleatorio
     • GET /facts ............. Lista paginada
     • GET /breeds ............ Razas de gatos
     
  ✅ Diseño moderno con Bootstrap 5
  ✅ Responsive (móvil, tablet, desktop)
  ✅ Navegación SPA con Angular Router
  ✅ Código limpio y documentado
  ✅ Listo para Netlify

🐛 SOLUCIÓN DE PROBLEMAS:

     npm install --force

  ❌ Puerto ocupado:
     npm start -- --port 4201

  ❌ Problemas con Node.js:
     El proyecto está configurado para Node 20+
